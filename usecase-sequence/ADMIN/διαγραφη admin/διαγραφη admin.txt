Λεκτική περιγραφή για περίπτωση χρήσης "ΔΙΑΓΡΑΦΗ ΛΟΓΑΡΙΑΣΜΟΥ" του admin

ΒΑΣΙΚΗ ΡΟΗ
1. Στον admin εμφανίζεται η σελίδα "ΔΙΑΓΡΑΦΗ ΛΟΓΑΡΙΑΣΜΟΥ"
2. Ο admin επιλέγει "ΔΙΑΓΡΑΦΗ".
3. Το σύστημα ελέγχει την αίτηση.
4. Εμφανίζεται μήνυμα: "Επιτυχής Διαγραφή".
5. Η Διαγραφή γίνεται με επιτυχία. Ο admin  διαγράφεται.
6. Ο admin μεταφέρεται στην σελίδα “ΑΡΧΙΚΗ ΣΕΛΙΔΑ”.

ΕΝΑΛΛΑΚΤΙΚΕΣ ΡΟΕΣ

Εναλλακτική Ροή 1
2α1. Ο admin επιλέγει "ΑΚΥΡΟ".
2α2. Το Σύστημα ελέγχει την αίτηση.
2α3. Ο admin μεταφέρεται στην σελίδα "ΕΠΕΞΕΡΓΑΣΙΑ ΛΟΓΑΡΙΑΣΜΟΥ".

Εναλλακτική Ροή 2
1α1. Εμφανίζεται μήνυμα “Αδυναμία φόρτωσης σελίδας”.
1α2. Ο χρήστης μεταφέρεται στη σελίδα “ΕΠΕΞΕΡΓΑΣΙΑ ΛΟΓΑΡΙΑΣΜΟΥ”.
