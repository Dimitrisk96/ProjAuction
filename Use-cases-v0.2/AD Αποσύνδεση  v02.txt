Λεκτική Περιγραφή για την Περίπτωση Χρήσης "ΑΠΟΣΥΝΔΕΣΗ" του admin.

ΒΑΣΙΚΗ ΡΟΗ 

1. O admin επιλέγει "ΑΠΟΣΥΝΔΕΣΗ" στην σελίδα "ΑΡΧΙΚΗ ΣΕΛΙΔΑ ADMIN".

2. Στον admin εμφανίζεται η σελίδα "ΣΕΛΙΔΑ ΑΠΟΣΥΝΔΕΣΗΣ".

3. Ο admin επιλέγει "ΕΞΟΔΟΣ".

4. Εμφανίζεται μήνυμα "Η αποσύνδεση σας έγινε με επιτυχία!".

5. Ο admin μεταφέρεται στην σελίδα "ΑΡΧΙΚΗ ΣΕΛΙΔΑ".



ΕΝΑΛΛΑΚΤΙΚΕΣ ΡΟΕΣ

Εναλλακτική Ροή 1
3α1. O admin επιλέγει "ΠΙΣΩ".

3α2. O admin μεταφέρεται στην σελίδα "ΑΡΧΙΚΗ ΣΕΛΙΔΑ ADMIN".
